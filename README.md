# Freight Quote

本地物流价格比价工具，支持：

- 赤道国际、纽酷国际、美琦国际报价上传
- 完全忽略隐藏 sheet
- 分层报错提示
- 优惠金额保存与纳入计算
- `批量查询`
- `多渠道比价`

## 环境要求

- Node.js 18+
- npm 9+

## 安装

```bash
npm install
```

## 启动

```bash
npm start
```

启动后打开：

```text
http://127.0.0.1:8787/freight-quote.html
```

## 使用方式

1. 左侧分别上传各物流商报价表
2. 如有临时优惠或附加费，在对应物流商卡片里填写`优惠金额`
3. `批量查询`里选择物流商和交货仓库，把 Excel 中的一列仓库代码直接粘贴到首列，然后点击`计算`
4. `多渠道比价`里输入单个仓库代码，查看各家物流商按升序排列的渠道结果

说明：

- 正数优惠金额会在原单价基础上减去该值
- 负数优惠金额会抬高最终价格
- 隐藏 sheet 会从识别到计算全程忽略
- 如果上传的新模板解析失败，会清空该物流商旧数据，并返回分层错误提示

## 健康检查

```bash
curl http://127.0.0.1:8787/health
```

返回：

```json
{"ok":true}
```

## 运行数据

运行过程中会自动生成这些本地数据文件：

- `data/freight/current.json`
- `data/freight/discounts.json`
- `data/freight/uploads/`

这些文件已经加入 `.gitignore`，不会进入仓库。
